# How to Use: UX Designer

## Quick Prompts

### User Research
- "Conduct user research to understand target audience needs"
- "Create user personas based on research data"
- "Design user journey maps for key workflows"
- "Perform competitive analysis of similar products"

### Information Architecture
- "Design information architecture for complex application"
- "Create site map and navigation structure"
- "Organize content hierarchy for optimal findability"
- "Design taxonomy and categorization system"

### Wireframing & Prototyping
- "Create low-fidelity wireframes for rapid iteration"
- "Build interactive prototype for usability testing"
- "Design user flows and interaction patterns"
- "Develop clickable mockups for stakeholder feedback"

### Usability Testing
- "Plan and conduct usability testing sessions"
- "Create testing scripts and scenarios"
- "Analyze user feedback and identify pain points"
- "Generate actionable insights from test results"

### Interaction Design
- "Design intuitive interaction patterns"
- "Create micro-interactions for better feedback"
- "Design gesture controls for mobile apps"
- "Build progressive disclosure patterns"

### Accessibility & Inclusion
- "Design inclusive experiences for all users"
- "Create accessible navigation patterns"
- "Implement WCAG compliance in design"
- "Design for users with disabilities"

### User Flow Optimization
- "Optimize checkout flow for e-commerce"
- "Simplify onboarding process for new users"
- "Design efficient task completion paths"
- "Reduce cognitive load in complex interfaces"

### Content Strategy
- "Develop content strategy for user engagement"
- "Write microcopy and interface text"
- "Create error messages and help text"
- "Design empty states and onboarding content"

### Collaboration
- "Work with UI designer on visual implementation"
- "Collaborate with product manager on requirements"
- "Partner with developers on feasibility"
- "Present designs to stakeholders"

### Metrics & Analytics
- "Define UX metrics and KPIs"
- "Analyze user behavior with analytics"
- "Measure task completion rates"
- "Track user satisfaction scores"

## Overview
The UX Designer agent specializes in user experience design, focusing on user research, information architecture, interaction design, usability testing, and creating intuitive user-centered designs.
