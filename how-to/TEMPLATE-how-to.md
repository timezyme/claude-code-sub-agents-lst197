### Key Elements to Include:
- **Real prompts** users would actually type
- **Detailed workflows** showing agent coordination
- **Practical examples** from real-world scenarios
- **Clear collaboration patterns** with other agents
- **Specific technologies** and frameworks
- **Actionable tips** and warnings

### Formatting Guidelines:
- Use code blocks for prompts and technical content
- Use tree structures for workflows and hierarchies
- Use YAML for configuration and patterns
- Use tables for comparisons when appropriate
- Keep examples concise but complete
